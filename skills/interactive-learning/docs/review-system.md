> **本文档说明基于艾宾浩斯遗忘曲线的复习机制。**

---

## 理论基础

### 艾宾浩斯遗忘曲线

不复习的情况下，记忆会随时间快速衰退：

| 时间 | 记忆保持率 | 遗忘率 |
|-----|-----------|-------|
| 20分钟 | 58% | 42% |
| 1小时 | 44% | 56% |
| 1天 | 33% | 67% |
| 2天 | 28% | 72% |
| 6天 | 25% | 75% |
| 31天 | 21% | 79% |

### 复习间隔策略

通过在关键节点复习，可以将记忆保持率提升到 90% 以上：

| 复习次数 | 间隔时间 | 说明 |
|---------|---------|------|
| 第1次 | 20分钟后 | 课程结束后立即复习 |
| 第2次 | 1小时后 | 短期记忆巩固 |
| 第3次 | 1天后 | 转为长期记忆 |
| 第4次 | 2天后 | 强化记忆 |
| 第5次 | 6天后 | 深度巩固 |
| 第6次 | 31天后 | 长期保持 |

```
用户说"复习"
    │
    ├── 读取 复习计划.md
    │
    ├── 获取今日待复习课程
    │
    ├── 无待复习？

    │
    ├── 有待复习 → 选择一门课程
    │   │
    │   ├── 1. 快速回顾
    │   │   ├── 显示课程小结
    │   │   └── 回顾检查站对话记录
    │   │
    │   ├── 2. 抽查提问
    │   │   ├── 随机选择检查站问题
    │   │   └── 用户回答 → AI 追问验证
    │   │
    │   ├── 3. 结果评估
    │   │   ├── ✅ 通过 → 安排下次复习
    │   │   └── ⚠️ 需加强 → 安排短期内重新复习
    │   │

    └── 继续下一门待复习课程
```

## 复习结果判定

| 结果 | 说明 | 下次复习安排 |
|-----|------|-------------|
| ✅ 通过 | 回答正确，理解清晰 | 按正常间隔安排下次复习 |
| ⚠️ 需加强 | 回答模糊，部分遗忘 | 1天后重新复习（不计入次数） |
| ❌ 需重学 | 完全遗忘，无法回答 | 建议重新学习该课程 |

## 复习内容设计

### 1. 快速回顾（2-3分钟）

显示课程小结，唤醒记忆：

```markdown
## 📖 快速回顾：01_什么是股票

**本节小结：**
- 股票是公司所有权的凭证
- 买入股票 = 成为公司股东
- 股东权利：分红权、投票权、剩余财产分配权
- 股价 = 市场对公司未来价值的预期

**上次复习：** 1小时前 ✅ 通过
```

### 2. 抽查提问（3-5分钟）

从检查站问题中随机抽取 1-2 个问题：

```markdown
## 🎯 抽查提问

**问题**：股票和债券有什么区别？

请简要回答，或说"跳过"进入下一个问题。
```

### 3. 结果反馈

```markdown
## ✅ 复习完成

**结果**：通过
**下次复习**：2024-01-17（1天后）

继续保持！你已完成本课 2/6 次复习。
```

3. 提示用户：

```markdown

> 第一复习时间：20分钟后（约 10:40）
>
> 建议：休息一下，20分钟后说"复习"巩固记忆。
```
